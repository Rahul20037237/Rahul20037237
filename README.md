### ūüß† Ambiguity-Aware Decision Agent
> **LLM ¬∑ BERT ¬∑ Reinforcement Learning ¬∑ MDP ¬∑ Information Theory**

An intelligent decision agent that determines whether to **act directly** or **ask for clarification** based on semantic uncertainty ‚ÄĒ combining NLP, information theory, and RL into a unified decision framework.

- **State** `s = (H, n, c, P(intent))` ‚ÄĒ entropy, steps, confidence, intent distribution
- **Entropy-based uncertainty**: `H(P) = -ő£ P(i) log P(i)` gates the ASK vs ACT decision
- **Reward shaping**: `R(ACT) = ¬Ī1` on correctness; `R(ASK) = -őĽ` to penalize over-caution
- **Q-learning** via Bellman: `Q*(s,a) = R(s,a) + ő≥ max Q*(s',a')` optimizes long-run decision quality
- Pipeline: Input ‚Üí LLM embeddings ‚Üí BERT classifier ‚Üí Softmax ‚Üí Entropy ‚Üí MDP ‚Üí Output

> Combines **NLP + Information Theory + Reinforcement Learning** to move from raw prediction to principled decision-making under ambiguity.
